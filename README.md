# Iris Dataset Exploratory Data Analysis (EDA)

This project contains a simple Python script that performs basic Exploratory Data Analysis (EDA) on the classic Iris dataset. It uses Pandas for data manipulation and Matplotlib to generate various foundational data visualizations.

## 🚀 Features

The script automatically generates four key visualizations to help understand the distribution and relationships within the Iris dataset:

1. **Flower Count (Bar Chart):** Displays the number of samples available for each Iris species.
2. **Sepal Length Distribution (Histogram):** Shows the frequency distribution of sepal lengths across all flowers.
3. **Sepal vs. Petal Length (Scatter Plot):** Visualizes the correlation between sepal length and petal length.
4. **Width Comparison (Scatter Plot):** Visualizes the correlation between sepal width and petal width.

## 🛠️ Technologies Used

- **Language:** Python 3.x
- **Libraries:** - `pandas` (for data loading and manipulation)
  - `matplotlib` (for creating the visualizations)
